Данная папка используется для настройки embedded-glassfish, который, в свою
очередь, используется для тестирования EJB компонентов.

Папка содержит два каталога: domains и lib.

Каталог domains должен содержать подкаталог domain1, который должен содержать папку 
config. Эту папку можно скопировать из настроенной инсталляции вашего 
рабочего сервера Glassfish.
Используйте скрипт install.sh в каталоге domain1.

Каталог lib должен содержать подкаталог install, который должен содержать
подкаталог applications. В каталоге applications должны быть папки
  __cp_jdbc_ra
  __dm_jdbc_ra
  __ds_jdbc_ra
  __xa_jdbc_ra
вашего настроенного рабочего сервера Glassfish.
Используйте скрипт install.sh в каталоге lib/install.
